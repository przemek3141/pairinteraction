26 /** \brief %Configuration storage
27  *
28  * This class stores the system configuration. In essence it is a
29  * wrapper around a `std::map` which `std::string` key and
30  * `Configuration::value` value. The wrapped value class has some
31  * advanced features which make it better than a pure `std::string`
32  * value.
33  */

136  /** \brief Append operator
137  *
138  * This will append the right hand side to the left hand side. If
